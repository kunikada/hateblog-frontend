# デプロイ手順（最小）

## 目的
- フロントエンドの最小デプロイ手順のみを示す
- NginxのDockerイメージでdistを配信する構成に限定する

## 対象範囲
- ビルド成果物の生成
- Nginxコンテナでの静的配信
- 最小限の確認

## 前提
- Node.jsとpnpmが利用可能
- Viteでビルドしdistを生成
- SPAとして配信する

## 手順
- 依存関係をインストールする
- ビルドを実行してdistを生成する
- distをNginxイメージに配置する
- 配信確認を行う

## 配信
- Nginxの公式イメージを使う
- distを /usr/share/nginx/html に配置する
- 静的ファイルが配信できることを確認する

## 最小確認
- トップページが表示される
- 主要ルートが表示される
- 静的アセットが読み込まれる

## 失敗時
- 直前の安定版イメージへ切り戻す
- ビルドログとdistの内容を確認する

## 運用メモ
- イメージタグにコミットSHA等を付ける
- latest運用は避ける
