# 404エラーページ

## 画面概要
- **画面名**: 404エラーページ
- **URL**: 存在しないパスにアクセスした場合に表示
- **目的**: ユーザーにページが見つからないことを通知し、適切な画面へ誘導する

## レイアウト構成

### ヘッダー
表示する

### フッター
なし

### サイドバー
なし

### メインコンテンツ
- エラーアイコン（または404イラスト）
- エラーメッセージ: "ページが見つかりません"
- 説明文: "お探しのページは存在しないか、移動した可能性があります"

## 表示要素

### エラー表示
- 404: 大きなフォントサイズで表示
- アイコン: 検索やファイルが見つからないイメージ
- メッセージ: シンプルで分かりやすい文言

### ナビゲーション
- グローバルナビゲーションのみ
