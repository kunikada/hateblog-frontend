# 月次ランキング

## 画面概要
- **画面名**: 月次ランキング
- **URL**: `/rankings/:year/:month`
- **URLパラメータ**:
  - `year`: 対象年（YYYY形式）
  - `month`: 対象月（MM形式、1-12）
  - `limit`: 取得件数（オプション、デフォルト: 100）
- **目的**: 指定年月のブックマーク件数上位エントリーTOP 100を表示

## 年次ランキングとの差分

### 主な違い
1. **対象期間**: 月単位
2. **APIエンドポイント**: `/api/v1/rankings/monthly`
3. **ページタイトル**: "2025年1月のランキング TOP 100"
4. **週次ランキングリンク**: 各週へのクイックリンク

### 共通部分
- ランキングエントリーカード構成
- インタラクション
- 状態管理
- デザインガイドライン

## レイアウト構成

### パンくずリスト
- ランキング > 2025年 > 1月

### ページヘッダー
- ページタイトル: "2025年1月のランキング TOP 100"
- 年月選択ナビゲーション
  - 前月ボタン
  - 年月表示（ドロップダウン）
  - 翌月ボタン

### ヒーローセクション（オプション）
- 月間トップ3のエントリーを大きく表示
- 年次ランキングと同じ構成

### 週次ランキングリンク
- 各週へのクイックリンク
  - "第1週のランキング" 〜 "第4-5週のランキング"
  - PC: 水平スクロール
  - SP: 横スクロールチップ

### メインコンテンツ
- ランキングエントリーリスト（4位〜100位）
- 年次ランキングと同じ構成

## API連携

### 初期表示（月次ランキング取得）
```
GET /api/v1/rankings/monthly?year={year}&month={month}&limit={number}
```

#### リクエストパラメータ
- `year`: 対象年（YYYY形式、必須）
- `month`: 対象月（MM形式、1-12、必須）
- `limit`: 取得件数（オプション、デフォルト: 100、最大: 100）

#### レスポンス
```json
{
  "period_type": "monthly",
  "year": 2025,
  "month": 1,
  "week": null,
  "entries": [
    {
      "rank": 1,
      "entry": {
        "id": "uuid",
        "title": "記事タイトル",
        "url": "https://example.com/article",
        "posted_at": "2025-01-15T10:30:00Z",
        "bookmark_count": 1234,
        "excerpt": "記事の抜粋...",
        "tags": [...],
        "favicon_url": "..."
      }
    },
    // ...
  ],
  "total": 5000
}
```

## インタラクション

### 年月選択
- 前月ボタン: `year` と `month` を調整してページ再読み込み
- 翌月ボタン: `year` と `month` を調整してページ再読み込み
- 年月ドロップダウン: 年月を選択してページ再読み込み

### 週次ランキングリンククリック
- `/rankings/{year}/week/{week}` に遷移
