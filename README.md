# hateblog-frontend

はてなブックマークのエントリーを閲覧・検索できるWebアプリケーションのフロントエンドです。

## 主な機能

- 新着・人気エントリーの一覧表示
- アーカイブ（年/月/日）とランキング（年/月/週）
- キーワード検索・タグ別一覧
- はてブ数によるフィルタリング
- 閲覧履歴（ローカル保存）

## 技術スタック

- React 19 + Vite
- TanStack Router + TanStack Query
- Tailwind CSS
- OpenAPI Generator（API型生成）

## セットアップ

```bash
pnpm install
```

## 開発

```bash
pnpm dev
```

## ビルド

```bash
pnpm build
```
